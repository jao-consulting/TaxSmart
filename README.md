# TaxSmart

The French Tax Calculator is an open-source project hosted on GitHub designed to simplify and streamline the complex
process of calculating taxes in France.

## Prerequisites

### Backend

- Java >= 21
- Maven >= 3.9.5
- Spring Boot >= 3.2.0

### Frontend

- Node >= 20
- Nuxt 3
